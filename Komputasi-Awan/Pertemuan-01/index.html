Pekan 1: Persiapan Lingkungan dan Pengenalan Dasar

Mata Kuliah Komputasi Awan

🔄 Deskripsi Singkat

Pada pekan pertama, mahasiswa menyiapkan seluruh lingkungan pengembangan: instalasi Python, Node.js, PostgreSQL, dan Docker. Selain itu, mempelajari konsep dasar containerization dan alur kerja pengembangan aplikasi.

Ilustrasi: Infrastruktur Cloud

🎯 Capaian Pembelajaran

  • Mampu menginstal dan mengonfigurasi Python, Node.js, PostgreSQL, dan Docker.
  • Memahami alur kerja dan struktur direktori proyek berbasis Flask, React.js, dan Docker.

💻 Persiapan Lingkungan Backend

  • Instalasi Python dan konfigurasi lingkungan pengembangan Flask.
  • Setup awal aplikasi backend menggunakan Flask.

Ilustrasi: Instalasi Python & Flask

🌐 Persiapan Lingkungan Frontend

  • Instalasi Node.js sebagai runtime dan manajemen paket.
  • Persiapan pengembangan aplikasi frontend dengan React.js.

Ilustrasi: Instalasi Node.js & React.js

🗄️ Konfigurasi Database (PostgreSQL)

  • Instalasi PostgreSQL dan setup database awal.
  • Panduan konfigurasi koneksi aplikasi dengan database.

Ilustrasi: Konfigurasi PostgreSQL

🐳 Pengenalan dan Instalasi Docker

  • Memahami konsep containerization untuk isolasi aplikasi dan dependensinya.
  • Langkah instalasi Docker dan verifikasi instalasi dengan docker --version.

Ilustrasi: Instalasi Docker

🔄 Workflow Pengembangan Proyek

  • Proses mulai dari coding, testing, hingga deployment.
  • Integrasi antara Flask, React.js, PostgreSQL, dan Docker.
  • Containerization dan orchestration untuk deployment yang efisien.

Ilustrasi: Workflow Pengembangan

📁 Struktur Direktori Proyek

Struktur direktori yang terorganisir memudahkan manajemen proyek:

├── backend/     # Aplikasi Flask
├── frontend/    # Aplikasi React.js
└── docker/      # Konfigurasi Docker & docker-compose.yml
