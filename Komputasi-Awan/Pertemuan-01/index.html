Pekan 1: Persiapan Lingkungan dan Pengenalan Dasar

🔧 Mengapa Persiapan Lingkungan?

  • Instalasi Python, Node.js, PostgreSQL, dan Docker sebagai fondasi untuk aplikasi cloud & containerization.
  • Menyusun workflow development yang efisien dari pengembangan hingga deployment.

Ilustrasi: Infrastruktur Cloud

🌐 Konsep Dasar Containerization

  • Mengisolasi aplikasi beserta dependensinya dalam container.
  • Memudahkan deployment ke berbagai lingkungan (development, staging, production).

Ilustrasi: Containerization

🔄 Workflow Pengembangan Proyek

  1. Persiapan environment (instalasi Python, Node.js, DB, Docker).
  2. Pengembangan backend menggunakan Flask dan frontend dengan React.js.
  3. Containerization & orchestration untuk deployment.

Ilustrasi: Alur Workflow

📝 Modul Praktikum

Materi praktikum mencakup instalasi, konfigurasi, dan pemahaman konsep dasar yang diperlukan untuk proyek cloud.

🔄 Deskripsi Singkat

Pada pekan pertama, mahasiswa menyiapkan seluruh lingkungan pengembangan: instalasi Python, Node.js, PostgreSQL, dan Docker, serta mempelajari dasar containerization dan alur kerja pengembangan aplikasi.

🎯 Tujuan Pembelajaran

  • Mampu menginstal dan mengonfigurasi Python, Node.js, PostgreSQL, dan Docker.
  • Memahami alur kerja dan struktur direktori proyek berbasis Flask, React.js, dan Docker.

👩‍💻 Pembagian Tugas Tim

  • Backend Engineer: Menyiapkan Python & Flask.
  • Infrastructure Engineer: Menyiapkan Docker & PostgreSQL.

Ilustrasi: Kolaborasi Tim

Instalasi Python


# Periksa versi Python & pip
python --version
pip --version

(Lihat panduan lengkap di modul praktikum.)

Instalasi Node.js


# Periksa versi Node.js & npm
node --version
npm --version

Instalasi PostgreSQL

Buat user dan database awal melalui pgAdmin atau CLI psql.

Instalasi Docker

Verifikasi instalasi dengan docker --version dan docker compose version.

📊 Kriteria Penilaian

  • Kesiapan lingkungan pengembangan (30%)
  • Pemahaman konsep dasar Docker (30%)
  • Kerapian struktur direktori (20%)
  • Instalasi berjalan tanpa error (20%)

Ilustrasi: Evaluasi & Penilaian

✅ Checklist Praktikum

  • Struktur direktori proyek (mis. cloud-project/) telah terbentuk.

Ilustrasi: Checklist Praktikum
