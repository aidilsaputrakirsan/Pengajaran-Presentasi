Pekan 1: Persiapan Lingkungan dan Pengenalan Dasar

Mata Kuliah Komputasi Awan

🔄 Deskripsi Singkat

Pada pekan pertama, kita akan mempelajari konsep dasar komputasi awan dan containerization. Materi ini memberikan gambaran umum mengenai peran teknologi cloud dalam pengembangan aplikasi modern. Fokusnya adalah pada pemahaman konsep, sedangkan aspek teknis akan dibahas di sesi praktikum.

Ilustrasi: Konsep Komputasi Awan

🎯 Capaian Pembelajaran

  • Memahami konsep dasar cloud computing dan containerization.
  • Mengenal peran teknologi dalam mendukung solusi aplikasi modern.
  • Memahami alur kerja pengembangan aplikasi secara umum.

💻 Pengenalan Lingkungan Backend

Backend adalah inti logika aplikasi yang mengelola data dan proses bisnis. Dalam konteks komputasi awan, backend membantu mengintegrasikan berbagai sistem dan memastikan alur data berjalan lancar.

Ilustrasi: Konsep Backend

🌐 Pengenalan Lingkungan Frontend

Frontend merupakan antarmuka yang berinteraksi langsung dengan pengguna. Konsep dasar pengembangan frontend melibatkan desain responsif, pengalaman pengguna (user experience), dan interaktivitas.

Ilustrasi: Konsep Frontend

🗄️ Pengenalan Konsep Database

Database adalah sistem penyimpanan dan pengelolaan data yang esensial bagi setiap aplikasi. Pendekatan manajemen data yang efisien mendukung operasi aplikasi yang skalabel dan andal.

Ilustrasi: Konsep Database

🐳 Pengenalan Containerization

Containerization adalah metode untuk mengemas aplikasi beserta dependensinya ke dalam satu paket yang konsisten. Konsep ini memungkinkan aplikasi berjalan dengan lancar di berbagai lingkungan.

Ilustrasi: Konsep Containerization

🔄 Alur Kerja Pengembangan Aplikasi

Pengembangan aplikasi modern melibatkan beberapa tahap: perencanaan, pengembangan, pengujian, dan deployment. Teknologi cloud mendukung integrasi dan efisiensi dalam setiap tahap tersebut.

Ilustrasi: Konsep Alur Kerja

📁 Prinsip Pengelolaan Proyek

Struktur proyek yang terorganisir memudahkan kolaborasi dan pemeliharaan kode. Prinsip dasar dalam pengelolaan proyek membantu tim bekerja secara efektif.

Ilustrasi: Manajemen Proyek

👩‍💻 Pembagian Peran dalam Tim

Setiap anggota tim memiliki peran yang spesifik untuk mencapai tujuan bersama. Contohnya, peran backend fokus pada logika aplikasi, sedangkan peran infrastruktur menangani lingkungan dan deployment.

Ilustrasi: Pembagian Peran

📝 Rangkuman & Diskusi

  • Memahami dasar-dasar cloud computing dan containerization.
  • Mengenal peran backend dan frontend dalam pengembangan aplikasi.
  • Menyadari pentingnya pengelolaan proyek dan pembagian peran tim.
