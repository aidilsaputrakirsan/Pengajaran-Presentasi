Pekan 1: Persiapan Lingkungan dan Pengenalan Dasar

🔧 Mengapa Persiapan Lingkungan?

  • Python, Node.js, PostgreSQL, dan Docker → fondasi untuk proyek cloud & containerization.
  • Membuat workflow development lebih efisien.

Ilustrasi: Infrastruktur Cloud

🌐 Konsep Dasar Containerization

  • Isolasi aplikasi & dependensi dalam container.
  • Memudahkan deploy di berbagai platform (dev, staging, production).

🔄 Workflow Pengembangan

  1. Persiapan environment (Python, Node.js, DB, Docker).
  2. Pengembangan backend Flask, frontend React.
  3. Containerization & Orchestration.

📝 Modul Praktikum

🔄 Deskripsi Singkat

Pada pekan pertama, mahasiswa mempersiapkan seluruh lingkungan pengembangan: instalasi Python, Node.js, PostgreSQL, dan Docker. Selain itu, mempelajari konsep dasar containerization dan workflow pengembangan secara keseluruhan.

🎯 Tujuan Pembelajaran

  • Menginstal dan mengonfigurasi Python, Node.js, PostgreSQL, dan Docker.
  • Memahami alur kerja dan struktur direktori proyek (Flask, React.js, Docker).

👩‍💻 Pembagian Tugas Tim

  • Backend Engineer: Menyiapkan Python & Flask.
  • Infrastructure Engineer: Menyiapkan Docker & PostgreSQL.

🛠️ Langkah-Langkah Praktikum

Instalasi Python


# Contoh:
python --version
pip --version

(Detail langkah di modul)

Instalasi Node.js


node --version
npm --version

Instalasi PostgreSQL

Buat user & database awal di pgAdmin atau CLI psql.

Instalasi Docker

Cek dengan docker --version & docker compose version.

📊 Kriteria Penilaian

  • Kesiapan lingkungan (30%)
  • Pengetahuan dasar konsep Docker (30%)
  • Kerapian struktur direktori (20%)
  • Kemampuan instalasi tanpa error (20%)

✅ Checklist Praktikum

  • Python, Node.js, PostgreSQL, Docker terinstal
  • Struktur direktori cloud-project/ terbentuk

🙋‍♂️ Latihan Mandiri

  • Coba jalankan Hello World di Python & Node.js
  • Buat database tambahan di PostgreSQL

📢 Deadline & Pengumpulan

Kumpulkan bukti instalasi di LMS sebelum akhir Pekan 1.
