ğŸ Pengenalan Flask

Flask adalah framework web minimalis untuk Python yang memungkinkan pembuatan aplikasi web dan API dengan mudah. Flask dikenal karena kesederhanaannya dan fleksibilitasnya.

  • Mudah dipelajari dan digunakan
  • Tidak terlalu memaksakan struktur tertentu
  • Cocok untuk aplikasi kecil hingga menengah

ğŸ—‚ï¸ Struktur Dasar Aplikasi Flask

Struktur aplikasi Flask yang sederhana umumnya melibatkan:

  • app.py sebagai file utama
  • Folder venv/ untuk virtual environment
  • File requirements.txt untuk daftar dependensi

Contoh kode dasar dalam app.py: 


from flask import Flask
app = Flask(__name__)

if __name__ == '__main__':
    app.run(debug=True)

ğŸ”€ Konsep Routing di Flask

Routing adalah mekanisme untuk menghubungkan URL dengan fungsi yang menangani permintaan. Dengan menggunakan decorator @app.route(), kita dapat menentukan endpoint dan metode HTTP (GET, POST, dll).

  • Mapping URL ke fungsi
  • Mendukung parameter URL untuk endpoint dinamis
  • Mendukung berbagai metode HTTP seperti GET dan POST

ğŸ’¡ Contoh Routing


@app.route('/api/hello', methods=['GET'])
def hello():
    return jsonify({'message': 'Hello, World!'})

Pada contoh di atas, ketika URL /api/hello diakses dengan metode GET, fungsi hello() akan mengembalikan respon berupa JSON.

ğŸ’» Contoh Kode API Sederhana


from flask import Flask, jsonify, request

app = Flask(__name__)

# Endpoint untuk menyapa umum
@app.route('/api/hello', methods=['GET'])
def hello():
    return jsonify({'message': 'Hello, World!'})

# Endpoint dengan parameter dinamis
@app.route('/api/greet/', methods=['GET'])
def greet(name):
    return jsonify({'message': f'Hello, {name}!'})

if __name__ == '__main__':
    app.run(debug=True)

Kode di atas menunjukkan pembuatan API sederhana dengan dua endpoint: satu untuk menyapa secara umum dan satu lagi untuk menyapa berdasarkan parameter nama yang diberikan di URL.
