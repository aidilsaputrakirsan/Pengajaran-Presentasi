Pekan 2: Membuat API Sederhana dengan Flask

🔄 Deskripsi Singkat

Pada pekan kedua, mahasiswa akan membuat API sederhana menggunakan Flask. Tahap ini menitikberatkan pada struktur dasar Flask, routing, dan menjalankan server lokal.

Gambar: Logo Flask

🎯 Capaian Pembelajaran

  • Memahami struktur folder dan best practice dasar dalam pembuatan API Flask.
  • Mampu membuat endpoint dasar dengan Flask.

🛠️ Struktur Dasar Flask & Routing

Flask adalah framework minimalis Python untuk web development. Materi ini mencakup pembuatan aplikasi, konfigurasi folder, dan penentuan routing untuk API.

💻 Contoh Kode API dengan Flask


from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/api/hello', methods=['GET'])
def hello():
    return jsonify({'message': 'Hello, World!'})

if __name__ == '__main__':
    app.run(debug=True)
