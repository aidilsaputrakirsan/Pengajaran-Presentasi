Pekan 2: Membuat API Sederhana dengan Flask

💡 Kenapa Flask?

  • Framework Python minimalis untuk membuat API cepat.
  • Struktur sederhana, cocok untuk microservices & prototyping.

🏗️ Virtual Environment

  • Penting untuk memisahkan dependensi antar proyek.
  • Gunakan venv atau conda.

“Lingkungan bersih, hati tenang.” 😁

🌐 Menjalankan Server Lokal

Jalankan app.run() di Flask → localhost:5000.

📝 Modul Praktikum

🔄 Deskripsi Singkat

Pekan ini, mahasiswa akan membuat API sederhana dengan Flask. Fokus pada struktur dasar Flask, routing, dan menjalankan server lokal.

🎯 Tujuan Pembelajaran

  • Memahami struktur folder dan best practice dasar pembuatan API Flask.
  • Mampu membuat endpoint dasar dengan Flask.

👩‍💻 Pembagian Tugas Tim

  • Backend Engineer: Membuat aplikasi Flask & routing dasar.
  • Infrastructure Engineer: Memastikan environment Python + venv jalan.

🛠️ Langkah-Langkah Praktikum

Membuat Virtual Environment


cd cloud-project/backend
python -m venv venv

Aktifkan dengan source venv/bin/activate (Mac/Linux) atau .\venv\Scripts\activate (Windows).

Menginstal Flask


pip install Flask

Membuat file app.py


from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/')
def home():
    return jsonify({"message": "Hello from Flask!"})

Menjalankan Aplikasi Flask


python app.py
# Akses http://localhost:5000/

📊 Kriteria Penilaian

  • Fungsi API Hello World berjalan (40%).
  • Kerapian kode & struktur proyek (30%).
  • Dokumentasi singkat (20%).

✅ Checklist Praktikum

  • Virtual environment terbuat & aktif.

  • Endpoint / menampilkan JSON Hello World.

🙋‍♂️ Latihan Mandiri
