Pekan 3: Membuat Aplikasi Frontend Sederhana dengan React + Vite

🔥 Mengapa React + Vite?

  • React: Library populer untuk UI dinamis.
  • Vite: Build tool cepat (hot reload, bundler minimalis).

Ilustrasi: React dan Vite

🏗️ Struktur Proyek React + Vite

  • Folder src untuk source code.
  • Folder public untuk aset statis.
  • File konfigurasi seperti vite.config.js.

Struktur Direktori React + Vite

🚀 Keuntungan Menggunakan Vite

  • Start up server sangat cepat.
  • Hot Module Replacement (HMR) yang efisien.
  • Bundling yang optimal untuk produksi.

📝 Modul Praktikum

🔄 Deskripsi Singkat

Mahasiswa akan membuat kerangka React menggunakan Vite untuk frontend. Tujuannya adalah membuat tampilan dasar dan menampilkan tulisan sederhana untuk memastikan React berhasil dijalankan dengan Vite.

🎯 Tujuan Pembelajaran

  • Memahami workflow pembuatan proyek React dengan Vite.
  • Mampu menjalankan aplikasi React di local development server menggunakan Vite.

👩‍💻 Pembagian Tugas Tim

  • Backend Engineer: Membantu menyesuaikan endpoint di Flask untuk uji coba fetch data.
  • Infrastructure Engineer: Menyiapkan Node.js environment dan setup React + Vite boilerplate.

Membuat Proyek React dengan Vite


    cd ../frontend
    npm create vite@latest my-react-app -- --template react
    cd my-react-app
    npm install
    npm run dev

Buka URL lokal (misal: http://127.0.0.1:5173/) di browser.

Struktur Direktori React + Vite


    frontend/
    └─ my-react-app/
        ├─ src/
        ├─ public/
        ├─ package.json
        ├─ vite.config.js
        └─ ...

Membuat Halaman Sederhana


// src/App.jsx
import React from 'react';


export default App;

Menjalankan Aplikasi React + Vite


    npm run dev

Pastikan tampilan "Hello from React + Vite!" muncul di browser.

📊 Kriteria Penilaian

  • Aplikasi React berjalan dan menampilkan tampilan sederhana (50%).
  • Struktur proyek dan kebersihan kode (30%).
  • Dokumentasi singkat (20%).

✅ Checklist Praktikum

  • Proyek React berhasil dibuat menggunakan Vite.
  • Tampilan "Hello from React + Vite!" muncul di browser.
  • Tidak ada error pada console browser.

🙋‍♂️ Latihan Mandiri

  • Tambahkan komponen baru (misalnya Header.jsx) dan tampilkan pesan berbeda.
  • Coba styling sederhana (CSS).
