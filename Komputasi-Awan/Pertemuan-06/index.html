Persiapan Awal

Pastikan Docker Desktop telah berjalan sebelum menjalankan perintah Docker.


    # Cek status Docker
    docker info

Membuat Dockerfile


    # backend/Dockerfile
    FROM python:3.9-slim

    WORKDIR /app

    COPY requirements.txt requirements.txt
    RUN pip install --no-cache-dir -r requirements.txt

    COPY . .

    EXPOSE 5000
    CMD ["python", "app.py"]

Menyiapkan requirements.txt


    flask
    flask-cors
    psycopg2-binary

Membangun Docker Image


    cd backend
    docker build -t flask-backend:1.0 .

Pastikan build berjalan tanpa error.

Menjalankan Docker Container


    docker run -d -p 5000:5000 --name flask-container flask-backend:1.0

Cek container berjalan dengan docker ps.

Verifikasi di Browser

Buka http://localhost:5000 untuk memastikan Flask API berjalan di dalam container.
