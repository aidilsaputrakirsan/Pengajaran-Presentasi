Persiapan Awal

Pastikan Docker Desktop telah berjalan sebelum menjalankan perintah Docker.


    # Cek status Docker
    docker info

Membuat Dockerfile untuk React dengan Vite


    # frontend/my-react-app/Dockerfile
    FROM node:14-alpine AS build

    WORKDIR /app

    COPY package*.json ./
    RUN npm install

    COPY . .
    RUN npm run build

    FROM nginx:stable-alpine
    COPY --from=build /app/dist /usr/share/nginx/html

    EXPOSE 80
    CMD ["nginx", "-g", "daemon off;"]

Membangun Docker Image


    cd frontend/my-react-app
    docker build -t react-frontend-vite:1.0 .

Pastikan build berjalan tanpa error dan folder dist dihasilkan.

Menjalankan Docker Container


    docker run -d -p 3000:80 --name react-container-vite react-frontend-vite:1.0

Cek container berjalan dengan docker ps.

Verifikasi di Browser

Buka http://localhost:3000 untuk memastikan aplikasi React berjalan di dalam container.
