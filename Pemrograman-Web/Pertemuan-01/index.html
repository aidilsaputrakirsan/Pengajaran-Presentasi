🎓 Pemrograman Web

Pertemuan 1: Pengenalan & Persiapan Lingkungan

Node.js • PostgreSQL • Git & GitHub • React (Vite) • Express.js

🎯 Tujuan Pembelajaran

  • Mampu menginstal & mengonfigurasi Node.js, PostgreSQL, pgAdmin.
  • Memahami penggunaan Git & GitHub untuk version control.
  • Mampu menyiapkan struktur folder dasar proyek (frontend & backend).

👨‍💻 Pengantar Node.js

  • Node.js adalah runtime JavaScript berbasis Chrome V8 Engine.
  • Memungkinkan eksekusi JavaScript di sisi server.
  • Ekosistem luas: NPM menyediakan ribuan modul.

🔌 Instalasi Node.js

  1. Unduh Node.js LTS via nodejs.org 
  2. 
# Setelah instalasi, cek versi:
node -v
npm -v

Pastikan versi Node dan NPM muncul di terminal/command prompt.

🐘 PostgreSQL & pgAdmin

  • PostgreSQL: Sistem Manajemen Basis Data Relasional (RDBMS) open-source.
  • pgAdmin: GUI untuk memudahkan administrasi database PostgreSQL.
  • Dapat mengelola schema, tabel, user, dan lainnya.

⚙️ Instalasi PostgreSQL & pgAdmin

  1. Unduh installer sesuai OS: postgresql.org/download
  2. Jalankan installer, set user dan password untuk postgres.
  3. Buka pgAdmin untuk kelola database, verifikasi service PostgreSQL berjalan.

🌱 Pengantar Git & GitHub

  • Git adalah version control system terdistribusi.
  • GitHub memudahkan hosting repo Git secara daring untuk kolaborasi.
  • Memungkinkan tim mengelola kode bersama, melacak perubahan, dan melakukan branching.

🌀 Langkah Dasar Git & GitHub

  1. Instal Git (cek git --version)
  2. Konfigurasi Global:
    git config --global user.name "Nama"
    git config --global user.email "email@example.com"
  3. Buat Repo GitHub & sambungkan remote: 
    
git init
git add .
git commit -m "Initial commit"
git remote add origin 
git push origin main

📁 Struktur Dasar Proyek


root-folder/
├─ frontend/
│   ├─ src/
│   └─ package.json
└─ backend/
    ├─ src/
    └─ package.json

Pisahkan tanggung jawab frontend & backend agar kode lebih terorganisir.

⚛️ Setup Frontend (React + Vite)


# Membuat proyek React via Vite:
npm create vite@latest my-frontend --template react

cd my-frontend
npm install
npm run dev

Halaman default akan terbuka di localhost:5173 (atau port lain).

⚙️ Setup Backend (Express.js)


mkdir my-backend
cd my-backend
npm init -y
npm install express pg cors

# Contoh sederhana server Express:
touch index.js

# index.js
const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello from Express!');
});

app.listen(port, () => {
  console.log(`Server running at http://localhost:${port}`);
});

Jalankan dengan node index.js atau npm start.
