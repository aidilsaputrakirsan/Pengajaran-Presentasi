Pertemuan 1: Pengenalan & Persiapan Lingkungan

🔧 Mengapa Persiapan Lingkungan?

  • Node.js: Runtime JavaScript di server.
  • PostgreSQL: Basis data relasional untuk menyimpan data.
  • Git & GitHub: Version control & kolaborasi tim.

Tanpa lingkungan yang tepat, proses development akan terhambat.

  • Frontend = Area restoran & menu (tampilan).
  • Backend = Dapur (logika bisnis & pengolahan data).
  • Database = Gudang (penyimpanan bahan baku).

📘 Modul Praktikum

📖 Deskripsi Singkat

Pekan ini, mahasiswa mempersiapkan lingkungan pengembangan: Node.js, PostgreSQL dan pgAdmin, Git & GitHub, serta struktur dasar proyek (frontend dengan React + Vite, backend dengan Express.js).

🎯 Tujuan Pembelajaran

  • Mampu menginstal & mengonfigurasi Node.js, PostgreSQL, pgAdmin.
  • Memahami penggunaan Git & GitHub.
  • Memiliki struktur folder dasar proyek (frontend & backend).

👥 Pembagian Tugas Tim

  • Frontend Engineer: Menyiapkan repository React + Vite di GitHub.
  • Backend Engineer: Menyiapkan repository Express.js di GitHub.

🛠️ Langkah-Langkah Praktikum

a. Instalasi Node.js

  1. Unduh & instal Node.js LTS dari nodejs.org.
  2. Cek dengan perintah berikut: 
    
node -v
npm -v

b. Instalasi PostgreSQL & pgAdmin

  1. Unduh PostgreSQL di postgresql.org/download
  2. Instal pgAdmin (bawaan installer PostgreSQL).
  3. Buat user & DB di pgAdmin:
    • User: postgres
    • Database: ecommerce-kecil

c. Persiapan Git & GitHub

  1. Instal Git: git-scm.com
  2. Buat akun GitHub & repository baru:
    • nama-proyek-frontend
    • nama-proyek-backend
  3. Clone repository ke local machine.

d. Inisialisasi Proyek Frontend (React + Vite)


npm create vite@latest ecommerce-frontend --template react

cd ecommerce-frontend
npm install

# Test jalankan
npm run dev

Lakukan commit & push: 


git add .
git commit -m "Initial commit (React + Vite)"
git push origin main

e. Inisialisasi Proyek Backend (Express.js)


mkdir nama-proyek-backend
cd nama-proyek-backend
npm init -y
npm install express pg cors

git add .
git commit -m "Initial commit (Express)"
git push origin main

🧑‍💻 Latihan Mandiri

  1. Coba buat satu branch terpisah di GitHub, lakukan commit & merge.

🛠️ Troubleshooting Tips

  • Node Tidak Terdeteksi: Cek PATH environment, buka terminal baru lalu cek node -v.
  • PostgreSQL Service Error: Pastikan service PostgreSQL berjalan.
  • GitHub Permission Denied: Biasanya karena SSH key atau username/password salah.
