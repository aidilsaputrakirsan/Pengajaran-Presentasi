🎓 Pemrograman Web

Pertemuan 1: Pengenalan & Persiapan Lingkungan

📖 Apa Itu Pemrograman Web?

Pemrograman web adalah proses membuat dan memelihara aplikasi berbasis web.

  • Frontend: Bagian yang dilihat dan berinteraksi dengan pengguna. Seperti tampilan restoran (dekorasi, menu).
  • Backend: Logika bisnis dan pengolahan data. Seperti dapur restoran tempat makanan disiapkan.
  • Database: Penyimpanan data. Seperti gudang bahan makanan.

🌐 Evolusi Teknologi Web

  • Web 1.0: Halaman statis, hanya membaca.
  • Web 2.0: Interaktif, berbagi informasi (media sosial).
  • Web 3.0: Terdesentralisasi, berbasis blockchain dan AI.

🔧 Teknologi Utama dalam Pemrograman Web

  • Frontend:
    • HTML, CSS, JavaScript.
    • Framework: React, Angular, Vue.js.
  • Backend:
    • Node.js, Express.js, Django.
  • Database:
    • Relasional: PostgreSQL, MySQL.
    • NoSQL: MongoDB, Firebase.

🏢 Peran Pemrograman Web dalam Industri

  • Aplikasi Utama:
    • E-commerce (Shopee, Amazon).
    • Media sosial (Instagram, Twitter).
    • Sistem internal perusahaan (ERP, CRM).
  • Keterampilan yang Dibutuhkan:
    • Adaptasi teknologi terbaru.
    • Kemampuan pemecahan masalah.

📘 Modul Praktikum

🎯 Tujuan Pembelajaran

  • 💻 Menginstal & mengonfigurasi Node.js, PostgreSQL, dan pgAdmin.
  • 📂 Memahami penggunaan Git & GitHub.
  • 📦 Memiliki struktur folder proyek dasar (frontend & backend).

🛠️ Langkah-Langkah Praktikum

1. Instalasi Node.js

node -v
npm -v

2. Instalasi PostgreSQL

  • Download: postgresql.org
  • Buat user: postgres
  • Buat database: ecommerce-kecil

3. Persiapan Git & GitHub

  • Instal Git: git-scm.com
  • Buat repository: frontend & backend

📊 Kriteria Penilaian

  • ✅ Kesiapan lingkungan (30%)

  • ✅ Kerja sama tim (30%)

✅ Checklist

  • Node.js & PostgreSQL terinstal
  • Database `ecommerce-kecil` tersedia
  • Repository GitHub terinisialisasi
  • Proyek React & Express telah berjalan
