Pertemuan 2: Dasar Express.js & Struktur Routing

Express.js • Server Sederhana • GET & POST • Struktur Routing

🎯 Capaian Pembelajaran

  • Mampu membuat server sederhana dengan Express.js.
  • Mampu mengimplementasikan struktur routing dasar (GET, POST).

👨‍💻 Pengantar Express.js

  • Express.js adalah framework Node.js yang ringan & fleksibel.
  • Membantu membuat API / layanan web secara cepat & terstruktur.
  • Didukung banyak middleware & memiliki ekosistem luas.

🛠️ Membuat Server Sederhana

Langkah dasar:

  1. Inisialisasi proyek: npm init -y
  2. Instal Express: npm install express
  3. Buat file index.js dan tuliskan kode berikut:

// index.js
const express = require('express');
const app = express();
const PORT = 3000;

app.listen(PORT, () => {
  console.log(`Server running on http://localhost:${PORT}`);
});

Jalankan dengan node index.js.

📌 Dasar Routing (GET & POST)

Tambahkan route di index.js:


// GET route
app.get('/', (req, res) => {
  res.send('Hello GET from Express!');
});

// POST route
app.post('/submit', (req, res) => {
  res.send('Data has been submitted!');
});
  • GET: Mengambil data (READ)
  • POST: Mengirim data (CREATE)

🔧 Middleware Singkat

Middleware memproses req sebelum sampai ke route handler. 


// Agar Express dapat memproses JSON di body:
app.use(express.json());

app.post('/submit', (req, res) => {
  console.log(req.body); // Data dari client
  res.send('Data has been submitted!');
});

Membantu parsing data & keamanan (CORS, dsb.)

🚀 Testing Routes (Postman/Insomnia)

  1. Gunakan Postman atau Insomnia.
  2. Buat request GET ke http://localhost:3000/.
  3. Buat request POST ke http://localhost:3000/submit, body JSON.
  4. Periksa respon: "Data has been submitted!"

📂 Struktur Routing


my-backend/
├─ routes/
│   └─ userRoutes.js
├─ index.js
└─ ...

Pisahkan file route (contoh: userRoutes.js) agar lebih terorganisir. 


// routes/userRoutes.js
const express = require('express');
const router = express.Router();

router.get('/', (req, res) => {
  res.send('User GET route');
});

router.post('/', (req, res) => {
  res.send('User POST route');
});

module.exports = router;
