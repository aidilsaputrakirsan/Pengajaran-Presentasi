Pertemuan 5: Integrasi Frontend & Backend (Fetch Data)

🌐 Mengapa Perlu Integrasi?

  • Frontend (React) butuh data real-time dari server (Express).
  • Menampilkan data dari DB (PostgreSQL) kepada user.

⚙️ Tools untuk Integrasi

  • Fetch API (bawaan browser) atau Axios (library).
  • Endpoint di Express harus mendukung GET/POST dsb.

📘 Modul Praktikum

📖 Deskripsi Singkat

React (Vite) melakukan request ke endpoint Express.js, menampilkan data produk dari PostgreSQL.

🎯 Tujuan Pembelajaran

  • Mampu melakukan HTTP request (GET) dari React ke Express.
  • Menampilkan data produk di UI React.

👥 Pembagian Tugas Tim

  • Frontend Engineer: Kode fetch/axios untuk data produk.
  • Backend Engineer: Pastikan endpoint /produk GET berjalan.

a. Instal Axios (Opsional)


npm install axios

b. Modifikasi ProdukList.jsx


import axios from 'axios';
useEffect(() => {
  axios.get('http://localhost:3001/produk')
    .then(res => setProduk(res.data))
    .catch(err => console.error(err));
}, []);

c. Pastikan Backend Berjalan


node index.js

d. Jalankan Frontend


npm run dev

📊 Kriteria Penilaian

  • React menampilkan data dari endpoint Express (50%).
  • Tidak muncul error CORS (25%).
  • Error handling minimal (25%).

✅ Checklist Praktikum

  • Endpoint /produk GET berjalan.
  • React menampilkan data dari DB.
  • Minimal error handling (console.error).

🧑‍💻 Latihan Mandiri

  1. Tambahkan form (POST) dari React ke Express.
  2. Styling tampilan dengan CSS.

📅 Deadline & Pengumpulan

  • Akhir Pekan 5
  • Screenshot data produk + Link GitHub

🛠️ Troubleshooting Tips

  • CORS Issue: app.use(cors()) di backend.
  • URL Endpoint Salah: Pastikan localhost:3001/produk.
  • Axios Error: Cek JSON format, server live?
